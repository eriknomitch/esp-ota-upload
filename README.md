# esp-ota-upload

## Dependencies

* (PlatformIO)[http://platformio.org/]

## Installation

### Repository

Clone this repository.

### Configuration
Create a file at `src/env.h` with the following contents:

```C
#define WIFI_ESSID    "YourWiFiEssid"
#define WIFI_PASSWORD "YourWiFiPassword"
```
